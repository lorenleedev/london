# 프로젝트 설명
프론트엔드 기술능력을 보여주기 위한 가상의 서비스입니다.

<br/><br/>

# 사용한 기술
### 프론트엔드  
프레임워크: Next.js with Typescript   
상태관리: Zustand, Tanstack-Query  
UI와 스타일: Ant Design, Vanila Extract or Tailwind-CSS  
테스트: Jest  
다국어지원: i18n  
<br/>
### 백엔드  
firebase

<br/><br/>

# 서비스 주요기능
- 회원가입/로그인  
- 게시글 CRUD

<br/><br/>

# 이 프로젝트를 통해 보여주고자 하는 기술능력
- 클린 코드 작성 능력 (응집도, 단일책임, 추상화)  
- 함수형 프로그래밍 능력  
- 실무 테스트 코드 작성 능력
